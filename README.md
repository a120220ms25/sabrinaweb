# Sabrina's Personal Portal

歡迎來到 Sabrina 的個人入口網站！這是一個展示我常用的 AI 工具以及個人作品的空間。網站設計採用簡約現代風格，並以柔和的玫瑰粉 (#D69396) 作為品牌主色調。

🔗 **線上瀏覽**: [https://a120220ms25.github.io/sabrinaweb/](https://a120220ms25.github.io/sabrinaweb/)

## ✨ 特色

- **AI 工具箱**: 精選並分類展示常用的 AI 工具（如 Claude, ChatGPT, Gemini 等）。
- **作品集展示**: 展示個人開發的專案與遊戲（如飲料王挑戰）。
- **課程推薦**: 分享優質的學習資源。
- **V16 簡約風格**: 
  - 質感暖灰背景 (#E8E4E1)
  - 玫瑰粉強調色 (#D69396)
  - 現代化卡片設計與微互動動畫

## 🛠️ 技術棧

- **HTML5**: 語意化標籤結構
- **CSS3**: 
  - CSS Variables (變數系統)
  - Flexbox & Grid 排版
  - 響應式設計 (RWD)
- **SVG**: 客製化 Logo 與圖標

## 🚀 如何執行

如果你想在本地端運行這個專案：

1. **Clone 專案**
   ```bash
   git clone https://github.com/a120220ms25/sabrinaweb.git
   ```

2. **進入目錄**
   ```bash
   cd sabrinaweb
   ```

3. **啟動本地伺服器** (使用 Python 或 Node.js)
   ```bash
   # Python
   python3 -m http.server 8080
   
   # 或 Node.js (http-server)
   npx http-server
   ```

4. **瀏覽網站**
   打開瀏覽器訪問 `http://localhost:8080`
